 Sistema de Biblioteca em Java

Projeto desenvolvido para praticar Programação Orientada a Objetos utilizando Java.

 Funcionalidades

- Cadastro de livros
- Consulta de informações dos livros
- Gerenciamento do acervo

 Tecnologias

- Java
- IntelliJ IDEA

 Conceitos aplicados

- Classes e Objetos
- Encapsulamento
- Construtores
- Relacionamento entre classes
